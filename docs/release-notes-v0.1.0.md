# Release Notes Draft: `v0.1.0`

## ErgometerApp `v0.1.0`

First public baseline release of ErgometerApp.

### Highlights
- Structured indoor cycling sessions with FTMS trainer control.
- Optional BLE heart-rate sensor support.
- Workout import from ZWO/XML with typed error feedback.
- Session view with live telemetry and workout profile chart.
- Post-session summary with planned and actual TSS.
- In-app workout editor MVP with ZWO export.

### Stability and UX
- Improved BLE reconnect/session behavior.
- Device picker scan stability improvements.
- Theme parity refinements for MENU, SESSION, SUMMARY, and WORKOUT EDITOR (dark/light).

### Open Source Baseline
- MIT license.
- Contribution guide, code of conduct, and security policy.
- Issue and pull request templates.
