# ErgometerApp

Android app for connecting to an FTMS-compatible indoor bike and a BLE heart rate monitor.

## Features
- FTMS Indoor Bike Data parsing
- Control Point handling (request control, target power, stop/reset)
- External HR strap support
- Session tracking and summary

## Requirements
- Android 13+ (minSdk 33)
- FTMS-compatible bike
- BLE HR monitor (optional)

This project is intentionally minimal and protocol-focused.
