# Code of Conduct

## Our Standard

We are committed to providing a respectful, harassment-free environment for everyone.

Examples of behavior that contributes to a positive environment:
- Being respectful in discussions and code reviews.
- Accepting constructive feedback.
- Focusing on technical substance instead of personal attacks.

Examples of unacceptable behavior:
- Harassment, insulting language, or discrimination.
- Trolling, personal attacks, or repeated unconstructive disruption.
- Publishing private information without explicit permission.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and other public communication channels related to this repository.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.
They may remove, edit, or reject comments, commits, code, issues, or other contributions that violate this policy.

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers via a private channel where possible (for example GitHub private security/contact channels).
